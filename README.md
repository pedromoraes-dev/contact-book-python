#  Agenda de Contatos em Python

O programa funciona via terminal e permite gerenciar contatos de forma simples.

## Objetivo

Projeto desenvolvido para praticar **Python e Programação Orientada a Objetos (POO)** durante os estudos de desenvolvimento de software.

---

## Funcionalidades

-  Adicionar contato
-  Listar contatos em ordem alfabética
-  Buscar contato pelo nome
-  Remover contato
-  Menu interativo no terminal

---

## Conceitos praticados

Este projeto foi desenvolvido para praticar conceitos importantes de Python:

- Classes
- Objetos
- Métodos
- Atributos
- Listas de objetos
- Estrutura de controle (`if`, `while`, `for`)
- Organização de código em múltiplos arquivos

---

## Tecnologias utilizadas

- Python 3
